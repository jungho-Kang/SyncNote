# 📝 SyncNote

![React](https://img.shields.io/badge/React-61DAFB?logo=react&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=white)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-06B6D4?logo=tailwind-css&logoColor=white)

**SyncNote**는 효율적인 노트 관리와 실시간 동기화를 지원하는 프로젝트입니다.
테스트 단계지만 직관적이고 깔끔한 UI/UX를 제공합니다.

---

## 📌 프로젝트 개요

| 항목              | 내용                                |
| ----------------- | ----------------------------------- |
| **Frontend**      | React + TypeScript                  |
| **Backend**       | Node.js / Express (예정)            |
| **실시간 동기화** | WebSocket, Firebase 등 (예정)       |
| **목표**          | 직관적인 UI/UX와 효율적인 노트 관리 |

---

## ⚡ 기능

### ✅ 현재 테스트 중

- 노트 작성 및 수정
- 노트 삭제
- 노트 목록 조회
- 임시 저장 및 자동 동기화

### 🚀 향후 계획

- 사용자 계정 및 로그인 기능
- 노트 공유 및 협업 기능
- 태그 및 카테고리 관리
- 다크 모드 지원

---

## 🛠 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/your-username/SyncNote.git

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속
