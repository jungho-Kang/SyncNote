# 📝 SyncNote

**SyncNote**는 효율적인 노트 관리와 실시간 동기화를 지원하는 프로젝트입니다.  
아직 테스트 단계에 있으며, 주요 기능과 사용 방법을 아래에서 확인할 수 있습니다.

---

## 📌 프로젝트 개요

- React + TypeScript 기반 프론트엔드
- Node.js / Express 기반 백엔드 (예정)
- 실시간 동기화 기능 (WebSocket, Firebase 등 예정)
- 직관적이고 깔끔한 UI/UX 제공

---

## ⚡ 기능

현재 테스트 중인 기능:

- 노트 작성 및 수정
- 노트 삭제
- 노트 목록 조회
- 임시 저장 및 자동 동기화

향후 계획:

- 사용자 계정 및 로그인 기능
- 노트 공유 및 협업 기능
- 태그 및 카테고리 관리
- 다크 모드 지원

---

## 🛠 설치 및 실행

1. 저장소 클론

```bash
git clone https://github.com/your-username/SyncNote.git .
```

2. 의존성 설치

```bash
npm install
```

3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:3000` 접속

## 📁 폴더 구조

```
SyncNote/
├─ src/
│  ├─ api/          # API 요청 관련
│  ├─ components/   # 재사용 컴포넌트
│  ├─ pages/        # 페이지 단위 컴포넌트
│  ├─ hooks/        # 커스텀 훅
│  └─ utils/        # 유틸 함수
├─ public/          # 이미지, 아이콘 등 정적 파일
├─ package.json
└─ README.md
```

## 🎨 기술 스택

- **Frontend:** React, TypeScript, Tailwind CSS, Zustand, Vercel
- **Backend:** Java Spring boot ,Render
- **DB:** Supabase
- **기타:** Axios, React Router, Swiper
